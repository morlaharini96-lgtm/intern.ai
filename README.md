# smart move intern
smart mov intern is a student-focused career guidance and internship support platform designed to help students identify skills, explore opportunities, and improve career growth.

Features

- Student profile analysis
- Internship recommendations
- Skill tracking
- Resume guidance
- Career roadmap suggestions
- User-friendly interface

Technologies Used

- HTML
- CSS
- JavaScript
- Firebase

Purpose of the Project

The main goal of this project is to support students who struggle to find the right internships and career direction.

Future Improvements

- AI-based career suggestions
- Interview preparation section
- Real-time internship updates
- Personalized learning recommendations

Project Status

Currently under development and continuously improving.
